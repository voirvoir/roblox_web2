BÀI 2: GIỚI THIỆU VỀ CODING

PHẦN 2 - VÒNG LẶP

Code vòng lặp

Platform phải liên tục biến mất và xuất hiện lại, với một vài giây giữa mỗi lần thay đổi. Không thể viết vô số lệnh gọi hàm ,may mắn thay, với vòng lặp while , bạn không cần phải làm như vậy.

Vòng lặp while chạy code bên trong nó miễn là câu lệnh sau while vẫn đúng. Vòng lặp cụ thể này cần phải chạy mãi mãi, vì vậy câu lệnh chỉ cần để là true

  • Tạo một vòng lặp while true ở cuối tập lệnh của bạn
    • local platform = script.Parent

    local function disappear()
    platform.CanCollide = false
    platform.Transparency = 1
    end

    local function appear()
    platform.CanCollide = true
    platform.Transparency = 0
    end

    while true do

    end

Chuyển đổi platform

Trong vòng lặp while, bạn sẽ cần viết code để đợi vài giây giữa nền tảng biến mất và xuất hiện lại

Hàm có sẵn wait có thể được sử dụng cho việc này. Trong ngoặc đơn, số giây để chờ là cần thiết: ví dụ wait(3)
⚠️ Dù bạn làm gì, đừng bao giờ tạo vòng lặp while true mà không bao gồm hàm wait, và đừng kiểm tra code của bạn trước khi bạn đưa vào! Nếu bạn không chờ đợi, trò chơi của bạn sẽ bị đóng băng vì Studio sẽ không bao giờ có cơ hội rời khỏi vòng lặp và làm bất cứ điều gì khác.
3 giây là điểm khởi đầu hợp lý cho khoảng thời gian giữa mỗi trạng thái platform

  1. Trong vòng lặp while, hãy gọi hàm wait có 3 trong dấu ngoặc đơn

  2. Gọi hàm disappear

  3. Gọi lại hàm wait với 3 trong dấu ngoặc đơn

  4. Gọi hàm appear
    5. local platform = script.Parent

    local function disappear()
    platform.CanCollide = false
    platform.Transparency = 1
    end

    local function appear()
    platform.CanCollide = true
    platform.Transparency = 0
    end

    while true do
    wait(3)
    disappear()
    wait(3)
    appear()
    end
Code cho nền tảng đã hoàn tất! Kiểm tra code của bạn và bạn sẽ thấy rằng nền tảng biến mất sau ba giây và xuất hiện lại ba giây sau trong một vòng lặp

Bạn có thể nhân bản platform này để che phủ khoảng cách rộng hơn, nhưng bạn sẽ phải thay đổi thời gian chờ trong mỗi tập lệnh, nếu không, tất cả các platform sẽ biến mất cùng một lúc và người chơi sẽ không bao giờ có thể vượt qua
